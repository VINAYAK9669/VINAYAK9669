> #### 2] IP Address Tracker üåç

 ***Description***<br/>
This project is a solution to the IP address tracker challenge on Frontend Mentor. The goal of the challenge is to create a responsive web application that allows users to view their own IP address on a map and search for information about other IP addresses or domains.
<br/>

 ***Flow & Implementations*** <br/>
 - **`Landing Page`** : On intial render this website access the user `IP ADDRESS` and this data we will given to the `ipify` api which deternmines the `Latitude` and `Longitude` of that IP address. Once we get the `Latitude` and `Longitude` we will pass these data to the `Leaflet` map then it will show the location in the map.
 - **`Input search bar`** : Once user add the valid `Ip Address` then the same process need to happen as discussed above.

 ***Tech Stack used*** <br/>
 *React.js | Tailwind | API | Leaflet Library | Ipify Geolocation API* <br/>
 
***Implemented*** <br/>
